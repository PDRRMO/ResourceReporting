# PDRRMO Resource Management System
## Executive Summary (One-Page)

---

### WHAT IS IT?
A digital map system (like Google Maps) that shows where all emergency equipment is located across Iloilo Province. 

**Simple Analogy:** Think of it as "Google Maps for Fire Trucks, Ambulances, and Rescue Boats."

---

### WHY DO WE NEED IT?

| Current Problem | Our Solution |
|-----------------|--------------|
| Don't know where equipment is | See everything on a live map |
| Paper records get lost | Digital, searchable database |
| Can't track if deployed | Real-time status updates |
| Takes hours to find resources | Find nearest equipment in seconds |
| No overview of readiness | Dashboard shows statistics instantly |

---

### WHO USES IT?
- **PDRRMO Staff** - Monitor all resources
- **Municipal Coordinators** - Update their equipment status
- **Provincial Leaders** - View readiness reports
- **Emergency Responders** - Find nearest available resources

### WHAT DATA DO WE NEED?

For Each Equipment:
1. **Name** (e.g., "Fire Truck 05")
2. **Type** (Fire, Ambulance, Boat, etc.)
3. **Quantity** (how many units)
4. **Location** (GPS coordinates + municipality)
5. **Status** (Ready / Deployed / Maintenance)
6. **Photo** (optional but recommended)

Iloilo City, Oton, Pavia, Leganes, Santa Barbara, Dumangas
